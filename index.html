FAQ

Is this system live on campus?

This is a concept web prototype. Actual integration requires campus infrastructure and sensor systems.

How would real-time data be updated?

Through IoT devices or student ID scanners connected to the cloud, feeding real-time occupancy data to this dashboard.

Can it send alerts when areas are full?

Yes. Future versions could include SMS or email notifications based on live occupancy thresholds.
